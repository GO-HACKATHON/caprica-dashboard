# Caprica Server

> A dashboard for Caprica Systems to monitor everything

# Usage

```sh
$ yarn dev
# or
$ npm dev
```

# Pre-requisites

Create your own GMaps API Key (here)[https://developers.google.com/maps/documentation/javascript/get-api-key
], we need it to monitor recent accidents

```sh
$ yarn
# or
$ npm install
```
